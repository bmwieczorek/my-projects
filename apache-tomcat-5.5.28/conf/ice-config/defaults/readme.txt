This folder contains zip file downloaded from ESSM (a.k.a. ICE) team site:
http://dahl.dev.sabre.com:8080/releases/9.4.0.5/release.html

It contains recommended configuration for ESSM (a.k.a. ICE). 
Our (SSI GUI) current configuration is made by modifying these "defaults".

BE AWARE that files in above ice-config directory may contain some changes 
when compared to default configuration, so please don't forget to include 
the changes when migrating to the newer version of the configuration 
(clearly replacing the files is not enough!).
